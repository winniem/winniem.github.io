Customizable Keyboard

The customizable keyboard was a self-sponsored capstone project I worked on with two of my friends. Inspired by the difficulties multilingual people face when switching keyboard languages we decided to investigate whether it would be possible to design a keyboard whose individual keys had customizable displays. This would allow for the display of the key to update according to its output when the keyboard layout was changed.

In the four months we were given we managed to create a prototype
